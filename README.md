## 介绍
> 一个电报(Telegram)开发框架，基于MTProto协议  
> 支持双卡双待，支持普通账号登录(手机号+验证码, session登录正在开发中)与机器人账号登录(Token)   
> 支持多账号，框架可以多开登录，也可以在一个框架内登录多个账号    
> 支持二次开发，基于事件推送的插件系统，目前仅支持易语言开发插件
